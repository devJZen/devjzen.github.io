## Problem

[TwoSum](https://leetcode.com/problems/two-sum/)

target을 입력하면 list의 요소에서 target값을 더해서 나오도록 풀이하고 싶었다.

너무 오랜만에 시작하는 코딩이어서 Claude와 풀이하고 Claude의 풀이를 복기하는 방식으로 풀어보려고한다.

## Solution

```python

class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        Keymap = {}

        for i, num in enumerate(nums):
            complement = target - num

            if complement in Keymap:
                return [Keymap[complement], i]
            Keymap[num] = i

```

또는 for문을 두번 써서  [i, j] 요소를 확인한다.
